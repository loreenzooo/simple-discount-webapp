# Simple Discount Web App

A simple ASP.NET Web Forms application that simulates a food ordering system. This project demonstrates basic C# backend logic integration with an ASP.NET frontend, handling event listeners, price calculations, and UI updates.

## üìã Description

This web application displays a menu of popular Filipino dishes (such as Pork Adobo, Sinigang, and Sisig). It allows users to view the price of items and calculates a 15% discount on specific purchases. The project serves as a practice implementation of:
* **Web Forms Structure:** connecting `.aspx` UI with `.aspx.cs` code-behind.
* **Event Handling:** Triggering C# functions from HTML buttons.
* **Basic Logic:** Performing arithmetic operations for totals and discounts.

## üöÄ Features

* **Interactive Menu:** Grid layout displaying food items and prices.
* **Price Calculation:** Updates the display with the selected item's price.
* **Discount Logic:** Applies a **15% discount** to the transaction amount and displays the final total.
* **Responsive UI:** Basic CSS styling using Flexbox and Grid for a clean layout.
